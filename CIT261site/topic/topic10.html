TOPIC 10: HTML5 Tags - Video, Audio, and Canvas

VIDEO: This tag only support MP4, WebM, and Ogg

   
   // Note: I removed the opening and close tag symbol.
    
    //controls attribute adds video controls, like play, pause, and volume.
    video class="videoSetting" controls
        source src="video/week6.mp4" type="video/mp4"
        // default message.
        Your browser does not support the video tag.
    /video




AUDIO: This tag only support MP3, Wav, and Ogg


   // Note: I removed the opening and close tag symbol.
    
    //controls attribute adds audio controls, like play, pause, and volume.
    audio class="audioSetting" controls
        source src="audio/risas.mp3" type="audio/mpeg"
        // default message.
        Your browser does not support the audio element.
    /video
