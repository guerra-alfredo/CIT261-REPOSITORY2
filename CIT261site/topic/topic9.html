1.- To start did you notice the background color, well we are using the onload method, so when the page is loaded the color changes automatically


//this allows to run this code before anything else
body onload="ChangeBody()"
    //JS
    function ChangeBody(){
        document.body.style.backgroundColor = "#F5F6CE";
    }
