3.- This example use Animation and transition to move some colored balloons, first of all by using transition I moved the balloons to the right, despite you could not see how it moves because I am using opacity property to hide this, you will notice that it start from the center, after that I am using animation join with some keyframes to move the balloons upwards

Press The button, this is using the click event



//the image of balloons has a class call box
.box {
    opacity: 0;
}

// I move the image to the right by using transition.   
.box.horizTranslate {
    transition: 0.1s;
    margin-left: 30%;    
    position: relative;
    
    //I move the image upwards by using animation.
    animation: myfirst 10s linear 1s  normal;
}


// I increase the opacity each time the image move upwards
@keyframes myfirst {
    0%   {left:0px; top: 0px; opacity:0.1; }
    20%  {left:0px; top:-150px; opacity: 0.3; }
    40%  {left:0px; top:-300px; opacity: 0.6; }
    60%  {left:0px; top:-450px; opacity: 1; }
    80%  {left:0px; top:-600px; opacity: 0; }
    100%  {left:0px; top:0px; opacity: 0;}
}
